## What's Inside

| Resource | Description |
|:---------|:------------|
| **Safety Snapshot** | 6 non-negotiable checks before every session |
| **Health Ladder** | 5-level diagnostic: status → gateway → health → audit → dashboard |
| **Remote Access** | Cheat sheet for SSH tunnel + Tailscale Serve |
