# OpenClaw Companion Pack

Free companion resource for the **OpenClaw Essentials** book series by Edward Izgorodin.

- **Safety Snapshot** (6 non-negotiables)
- **Health Ladder** (5-level diagnostic)
- **Remote Access** cheat sheet (SSH tunnel + Tailscale Serve)
- **Tiny Troubleshooting Fork**
- **Gateway Alive Card** (8-checkbox print-ready card)
- **Emergency procedures** pointer (Panic Card lives in OpenClaw Made Safe, Ch. 5)

## Books

- **Vol. 1 — Your First OpenClaw**: From Zero to Gateway Alive (Setup Guide)
- **Vol. 2 — OpenClaw Made Safe**: The flagship operator manual
