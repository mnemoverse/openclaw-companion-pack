## How to read this page

Each entry lists:

- **Book / Pack version** — which edition is affected
- **Location** — chapter and section
- **Issue** — what changed or broke
- **Fix** — what to do instead
- **Status** — open / fixed in version X

## Reporting an issue

If you find a command that no longer works, a broken link, or a factual error:

1. Check [official OpenClaw docs](https://docs.openclaw.ai) first — the command may have changed upstream.
2. If the issue is in our book or pack, [open an issue]({{ site.github.repository_url }}/issues/new).

Include:

- Which book or pack version you are using
- Which OpenClaw release you are running (`openclaw --version`)
- The exact command or section that does not work
- What you expected vs what happened
